== Description ==

A plugin for interfacing your Wordpress with Push Messenger, an application for receiving custom push notifications on your iPhone.
You never need check your blog manually again and again just looking for new comment/trackback. When new comment/trackback on your blog available, this plugin will send you a iPhone push notification about that.

== Installation ==

1. Install Push Messenger for your iPhone (http://ipush.me)
2. Sign up for a Push Messenger Account.
3. Upload the wp-PushMessenger files into the '/wp-content/plugins/' directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Enter the User ID and Secret Token into wp-PushMessenger settings page
